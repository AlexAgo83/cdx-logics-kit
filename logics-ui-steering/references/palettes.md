# Palette selection

## Rule order

1. Reuse the project's existing colors, tokens, CSS variables, and component palette if they exist.
2. If the project has no defined palette, pick one fallback set from this file.
3. Do not invent arbitrary color systems by default.

Colors should support hierarchy, not compete with it.
